# Neovim Cheatsheet

**:Tutor** open the tutorial

## Telescope

**:Telescope colorscheme** show installed colorschemes

## Explorer Neotree

**H**  show/hide hidden files
**d**  delete selected file/folder

## Kickstart

**\<C-y>** accept autocomplete suggestion
**\<C-n>** select next suggestion
**\<C-p>** select previous suggestion
